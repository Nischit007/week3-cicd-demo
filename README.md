# Week 3 CI/CD Demo

Simple CI/CD demonstration project.

## Files

### app.sh
Prints:
- Application version
- Hostname
- Current date
- Kernel version
- Operating system
- System architecture

### test.sh
Runs app.sh and validates that it exits successfully.

## Usage

Run application:

```bash
./app.sh
```

Run tests:

```bash
./test.sh
```
